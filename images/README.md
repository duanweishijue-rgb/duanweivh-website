## 📝 如何添加图片（3步完成）

### 第1步：准备图片
- 用 Photoshop / 美图秀秀 / 在线工具 将图片调整为上方表格中的尺寸
- 保存为 JPG 格式，文件名用英文小写（如 `hero-1.jpg`）

### 第2步：放入对应文件夹
- 把图片拖拽到对应的子文件夹中
- 确保文件名与上表完全一致

### 第3步：提交到 GitHub
```bash
cd /Users/duanwei/Documents/QoderCN/2026-07-03/chat-2/site
git add images/
git commit -m "添加网站图片"
git push origin main
```

---

## ⚠️ 注意事项

1. **文件名必须完全一致**（区分大小写）
2. **图片格式统一用 JPG**（体积小，加载快）
3. **图片放入前请先压缩**（推荐用 [TinyPNG](https://tinypng.com) 在线压缩）
4. 如果没有放图片，页面会显示渐变色背景作为兜底
